# ArteIA

Formulário de pedidos de artes integrado ao Supabase.

## Configuração

1. Crie o projeto no GitHub com estes arquivos.
2. Na Vercel, importe o repositório.
3. Configure as variáveis de ambiente:

- VITE_SUPABASE_URL
- VITE_SUPABASE_ANON_KEY

4. No Supabase, tenha as tabelas: clientes, pedidos, arquivos.
5. No Storage, tenha os buckets públicos: fotos, logos, previas, originais.
